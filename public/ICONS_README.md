# PWA Icons Guide

## Required Icons for itsmycv.be PWA

The following icon files need to be placed in the `/public` directory:

- `icon-192x192.png` - 192x192px icon
- `icon-256x256.png` - 256x256px icon  
- `icon-384x384.png` - 384x384px icon
- `icon-512x512.png` - 512x512px icon
- `favicon.ico` - Standard favicon

## How to Generate Icons

### Option 1: Use your logo
1. Take your logo from: `https://res.cloudinary.com/docxvgl2f/image/upload/v1762906548/itsmycv_mtrcbq.png`
2. Use an online tool like:
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator
3. Upload your logo and download the generated icons
4. Place them in the `/public` folder

### Option 2: Create manually
1. Create square PNG images with transparent backgrounds
2. Use the brand color #ff007a (pink) as the main color
3. Export at the required sizes listed above
4. Ensure icons are centered and have proper padding
